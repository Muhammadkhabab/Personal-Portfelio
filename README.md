# MK-PORTFOLIO-MAIN

A modern, responsive, and dark-themed personal portfolio built with **Flutter Web** by **Muhammad Khabab**.

## Highlights

- Fully responsive design for desktop, tablet, and mobile
- Smooth single-page navigation between sections
- Dark UI with modern portfolio styling
- Project showcase with carousel interaction
- Skills, pricing, and contact sections
- Social and professional profile integration
